🔄 Pipeline Steps
These steps run in order every time a video is made:
🔍 Research→ 🧠 Brainstorm→ ✍️ Script→ 🎙️ Voice→ 🎬 Video 🖼️ Thumb 🏷️ SEO→ 📤 Upload→ ✅ Verify
❓ What Each Step Does
🔍 Research
Scans Google Trends, YouTube, Hacker News & News RSS for what's trending right now
🧠 Brainstorm
6 AI agents simultaneously plan the hook, thumbnail, SEO strategy and story angle
✍️ Script
Writes a full 10–12 min script with hooks, pattern interrupts, and a strong CTA
🎙️ Voice
Microsoft Neural TTS — natural speech in English or Hindi, processed for clarity
🎬 Video
Fetches stock footage, adds zoom effects, background music, section titles
📝 Captions
Auto-transcribes the voice and burns styled subtitles into the video
🖼️ Thumbnail
AI-generated cinematic image + bold title text + emotion badge + branding
📤 Upload
Publishes to YouTube with title, description, chapter timestamps, and tags
✅ Verify
Fetches the video back from YouTube to confirm it's live and accessible

🛠️ Quick Actions
🚀 Setup Guide — 15 Minutes to First Video
Complete these steps once. Everything is free — no credit card.
1️⃣
Install System Dependencies
On Ubuntu/Debian (Oracle Free Tier, Hostinger, etc.):
sudo apt install -y python3 python3-pip ffmpeg
pip install -r requirements.txt
2️⃣
Get Gemini API Key (1 min, free)
→ Go to aistudio.google.com
→ Sign in with Google → Click Get API Key → Copy it
→ Open your .env file → Set GEMINI_API_KEY=AIza...
3️⃣
Get Groq Key (Optional — Faster AI) (1 min, free)
→ Go to console.groq.com → Sign up free → Create API Key
→ Set GROQ_API_KEY=gsk_... in your .env
Optional but recommended — makes the bot 5× faster and has 14,400 free req/day
4️⃣
Get Pexels Key (1 min, free)
→ pexels.com/api → Sign up → Get API Key → PEXELS_API_KEY=...
5️⃣
Get Pixabay Key (1 min, free)
→ pixabay.com/api/docs → Sign up → PIXABAY_API_KEY=...
6️⃣
Connect YouTube Channel (10 min)
1. → console.cloud.google.com → New Project
2. → APIs & Services → Enable YouTube Data API v3
3. → Credentials → OAuth 2.0 Client ID → Desktop App
4. → Copy Client ID & Secret into .env
5. → On any computer with browser: python auth_setup.py
6. → Copy printed YOUTUBE_REFRESH_TOKEN into .env
7️⃣
Test Everything
python main.py --setup-check
All green = ready. Then go to the 🩺 API Health tab in this dashboard to verify.
8️⃣
Make Your First Video!
python main.py --run-now — or use the ▶ Make Video tab above.
First video takes ~20–30 minutes. Watch the ⚡ Live Monitor tab to see exactly what's happening.
🖥️ Run 24/7 on Oracle Free Tier (Recommended)
Oracle Cloud offers a forever-free VM (2 CPU, 1GB RAM — enough for this bot).
Sign up at oracle.com/cloud/free → Create Ubuntu VM → SSH in → Upload files → Run as service:

sudo nano /etc/systemd/system/ytbot.service
# paste: [Service] ExecStart=/usr/bin/python3 main.py WorkingDirectory=/path/to/bot
sudo systemctl enable ytbot && sudo systemctl start ytbot
sudo journalctl -u ytbot -f # live logs
